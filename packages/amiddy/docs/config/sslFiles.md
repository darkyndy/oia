# sslFiles
Ability to provide own files for the certificate that will be used by the middleware.
The configuration applies to the proxy when _vhost.https_ is set to `true`.


## sslFiles.cert
- Data type: String
- Required: Yes
- Default value: N/A
- Details:
    - Path to the certificate file to use.


## sslFiles.key
- Data type: String
- Required: Yes
- Default value: N/A
- Details:
    - Path to the certificate file to use.
